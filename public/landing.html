Perguntas frequentes

Tem alguma dúvida?

O SureEdge é gratuito?
Sim, 100% gratuito. O SureEdge roda inteiramente no seu navegador — sem servidor, sem assinatura, sem surpresas. Seus dados ficam salvos localmente no seu dispositivo.
Meus dados são seguros?
Completamente. Nenhum dado é enviado para servidores externos. Todas as informações — operações, casas, caixa — ficam salvas exclusivamente no localStorage do seu navegador. Só você tem acesso.
Preciso usar o Google Sheets?
Não. Você pode registrar operações manualmente pelo painel, uma a uma. A integração com Google Sheets é opcional, mas facilita muito quem já tem uma planilha de controle.
Como funciona a sincronização automática?
Após conectar sua planilha, o SureEdge faz uma leitura automática toda vez que você abre o painel e a cada 60 segundos enquanto está aberto. Novas operações são importadas sem duplicar as existentes.
Quais tipos de aposta são suportados?
O SureEdge suporta Surebet, Delay e Duplo Green — os três modelos mais usados por apostadores profissionais. Também há suporte para 48 esportes, de futebol a e-sports.
Posso usar no celular?
Sim. O SureEdge é totalmente responsivo. A interface se adapta a qualquer tamanho de tela, desde desktop até smartphones, com layout otimizado para cada dispositivo.
